Detailed Description

class RawPV implements the Gamma API interface for process variables (PV).

when a new instance is created, RawPV connects to gamma service and attach to the specified process variable. Each access to the data is forwarded to gamma service. Only some settings are buffered local. The connection to gamma service will be destroyed, when the instance is terminated.

Remarks
for numeric value use TypedPV instead this implementation.
