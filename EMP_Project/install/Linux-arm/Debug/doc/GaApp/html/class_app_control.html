void AppControl::loadModules ( )

load modules defined in configuration file

all modules need to be described in configuration path "/AppConfig/ModuleList/". For each module there need to be a node named "Module" and contains the attributes "name" and "library".

attribute "name" defines the name of the module, attribute "library" give the name of the library. It is important, that the file "lib[libraryName].so" is stored in the project folder "bin"
