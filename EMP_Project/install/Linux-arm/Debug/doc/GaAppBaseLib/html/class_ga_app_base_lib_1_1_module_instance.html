Detailed Description

class ModuleInstance wrapped the instance pointer to an external application module derived from GaAppBaseLib::ModuleInterface

To create a ModuleInstance a GaAppBaseLib::LibInstance is needed. From this library the function createNewInstance is called. System calls are forwarded to the external module.
