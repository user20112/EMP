Detailed Description

container class for library instance.

this class load a shared library, and keep handle until the instance is deleted.

to create a new module from loaded library call LibInstance::createNewInstance . Each created instance need to deleted by calling deleteInstance . The count of actual existing instances can be requested by calling getInstanceCount
