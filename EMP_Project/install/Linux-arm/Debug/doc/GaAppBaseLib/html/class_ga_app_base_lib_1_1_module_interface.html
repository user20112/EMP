Remarks
don't add more input parameter, because the calling function wouldn't know them!

actual situation

the constructor is called during all modules will be loaded. There is no guarantee witch modules are active now. During the construction the module is only for its own and should not try to connect to other modules.

jobs to do here

  • parse xml configuration and setup this instance
  • create and connect to own gamma variables and scheduler slots
